### 快速开始
```
npm run serve
```

### 打包
```
npm run build
```


### 配置
1. 工程配置 [vue-cli 配置](https://cli.vuejs.org/config/).
2. 自定义主题: 项目内置了 `antd 1.x`, 如果要自定义主题，可在 `vue.config.js` 的 less 配置项 `modifyVars` 修改[相关变量](https://ant.design/docs/react/customize-theme-cn)
